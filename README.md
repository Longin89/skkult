# skkulturist.ru - Сайт спортзала

Веб-сайт спортзала, разработанный на самодельном PHP-фреймворке.

## Описание

Проект представляет собой полнофункциональный веб-сайт спортзала со скрытой админ-панелью для управления контентом. Сайт построен на любительском  MVC-фреймворке.

## Основные возможности

### Публичная часть
- **Главная** - основная информация о зале
- **О нас** - история спортзала
- **Новости** - новости, связанные с деятельностью зала
- **Расписание** - расписание занятий
- **Цены** - цены на услуги 
- **Контакты** - контактная информация

### Админ-панель
- Управление контентом всех обозначенных выше разделов
- Загрузка и управление изображениями для новостей и галереи сайта
- CRUD операции для практически всей информации на сайте
- Система аутентификации и авторизации (файлы отсутствуют в репозитории из соображений безопасности)

## Стек

- **Любительский PHP-фреймворк - [Ruah MVC framework](https://bitbucket.org/parhamcurtis/ruah-mvc-framework/src/master/)**
- **MySQL**
- **Memcached**
- **Bootstrap**
- **jQuery**
- **jQuery UI**

## Архитектура проекта

### Структура директорий

```
skkult/
├── app/                    # Приложение
│   ├── controllers/        # Контроллеры (публичные и админ)
│   ├── models/            # Модели
│   ├── views/             # Представления (виды)
│   ├── lib/               # Библиотеки (Memcached, утилиты)
│   └── *.json             # Конфигурационные файлы ACL и меню
├── core/                  # Ядро фреймворка
│   ├── Application.php    # Основной класс приложения
│   ├── Router.php         # Маршрутизация
│   ├── Controller.php     # Базовый класс контроллера
│   ├── Model.php          # Базовый класс модели
│   ├── View.php           # Класс представления
│   ├── DB.php             # Работа с базой данных
│   ├── Session.php        # Управление сессиями
│   ├── Cookie.php         # Работа с cookies
│   ├── Input.php          # Обработка входных данных
│   └── validators/        # Валидаторы форм
├── config/                # Конфигурационные файлы (отсутствует в репо)
├── css/                   # Стили
├── js/                    # JavaScript файлы
├── images/                # Статические изображения
├── uploads/               # Загруженные файлы
└── index.php              # Точка входа
```

### Внимание

Часть конфигурационных файлов и функционал регистрации отсутствует в репозитории из соображений безопасности
